# Ansible Galaxy Playbooks 🌍🌍🌍
It is a ready-made playbook and role repository for Ansible.
https://galaxy.ansible.com/ui/

# Playbook Search:
ansible-galaxy search mysql

# Playbook Apply:
ansible-playbook nginx-deploy.yml [Default hostfile]
ansible-playbook -i debians-hosts.ini nginx-deploy.yml [Custom hostfile]

# Useful Ansible Commands:
ansible-playbook playbook.yml --syntax-check [Check Playbook Syntax]
ansible-playbook playbook.yml --check [Dry Run Mode]
