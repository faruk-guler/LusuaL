##############################
## Ansible Galaxy Collection
##############################
Ansible Galaxy Collection is a structure that brings together and packages modules, roles, playbook examples, plugins, and other content in Ansible. This makes complex structures easier to manage and reusable.

ansible-galaxy collection init # Creates a new collection skeleton
ansible-galaxy collection list # Lists installed collections
ansible-galaxy collection build # Packages the collection
ansible-galaxy collection publish # Uploads to Galaxy
ansible-galaxy collection install <name> --upgrade # Install/update collection
